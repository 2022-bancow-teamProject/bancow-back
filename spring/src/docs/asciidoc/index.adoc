[[Manager]]
== Manager
[[Manager-Overview]]
=== Manager Overview
==== Manager Status 설명
|===
| managerStatus | 설명

| PENDING_EMAIL
| 회원가입 후에 받아지는 상태, 이메일 인증이 필요합니다.

| PENDING_SUPER
| 이메일 인증 후에 받아지는 상태, SUPER 계정의 승인이 필요합니다.

| ADMIN
| 모든 인증이 끝난 후에 로그인 할 수 있는 상태입니다.

| SUPER
| Manager의 권한을 바꿀 수 있는 SUPER 계정입니다.
|===

==== Manager 회원가입 로직
|===
| http Method | url | 기능 | managerStatus | description | request body

| post
| /api/register
| 회원 가입
| PENDING_EMAIL - 이메일 인증 필요
| 가입 때 적은 이메일로 토큰이 포함된 인증 링크 전송
| email, username, password, password2

| get
| /api/authentication/{token}
| 이메일 인증
| PENDING_SUPER - SUPER 계정의 승인 필요
| 이메일로 발급된 링크를 클릭하면 인증 완료
| X

| patch
| /api/statustoadmin?id={id}
| SUPER 계정의 승인
| ADMIN
| SUPER 계정은 statusadmin을 통해 해당 계정을 PENDING_SUPER 에서 ADMIN으로 변경
| X

| post
| /api/login
| 로그인
| ADMIN
| 토큰 발급하여 token db에 저장
| X

| post
| /api/logout
| 로그아웃
| -
| token db에서 삭제
| X
|===

==== Manager 비밀번호 변경 로직
|===
| http Method | url | 기능 | description | request body

| post
| /api/findmanager
| 비밀번호 초기화
| 해당 이메일로 토큰이 포함된 인증 링크 전송
| email, username

| get
| /api/authentication/findmanager/{token}
| 이메일 인증
| 이메일로 토큰이 포함된 링크를 클릭하면 인증 완료
| X

| patch
| /api/authentication/findmanager/{token}/change-password
| 비밀번호 변경
| password1, password2를 입력하고 일치하면 비밀번호 변경
| password1, password2
|===
